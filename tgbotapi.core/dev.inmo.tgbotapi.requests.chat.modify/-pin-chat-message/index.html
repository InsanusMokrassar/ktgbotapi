PinChatMessage

@Serializable
data class PinChatMessage(val chatId: ChatIdentifier, val messageId: MessageId, val businessConnectionId: BusinessConnectionId? = null, val disableNotification: Boolean = false) : ChatRequest, SimpleRequest<Boolean> , MessageAction, DisableNotification, OptionallyBusinessConnectionRequest

Use this method to add a message to the list of pinned messages in a chat. If the chat is not a private chat, the bot must be an administrator in the chat for this to work and must have the 'can_pin_messages' admin right in a supergroup or 'can_edit_messages' admin right in a channel.
